# AGENTS.md

This file supplements `CLAUDE.md` with guidance specific to OpenCode sessions.

## Commands

```bash
npm install        # copies MathJax + Mermaid from node_modules to src/vendor/ (postinstall)
npm run tauri dev  # pass a file: `-- test.md`
cd src-tauri && cargo test   # Rust tests only (no JS tests exist)

# focused Rust test
cd src-tauri && cargo test renders_readme_fenced_code_blocks
```

Version source of truth is `package.json`. `make release` syncs it to `src-tauri/Cargo.toml` and `src-tauri/tauri.conf.json`, then commits, tags `v<version>`, and pushes.

## Architecture

- **Frontend**: `src/index.html` + `src/main.js` (~1600 lines, single module). Settings, TOC, search, drag-drop all wired in `main.js`. No framework.
- **Backend**: `src-tauri/src/lib.rs` (setup, CLI, nav handler), `markdown.rs` (Comrak + Ammonia rendering pipeline), `settings.rs` (`~/.config/emede/settings.json`), `share.rs` (LAN sharing on port 7777 via tiny_http), `view_state.rs` (per-doc scroll state).
- **Image paths**: Rust resolves relative image src to absolute paths → frontend converts to `asset://` protocol via `convertFileSrc`.
- **Security**: CSP in `tauri.conf.json` is restrictive (`default-src 'self'`). Ammonia strips `<script>`, event handlers, `javascript:` URLs.

## Gotchas

- **GPU on Linux**: `apply_gpu_setting()` in `lib.rs` reads settings and sets `WEBKIT_DISABLE_DMABUF_RENDERER`/`WEBKIT_DISABLE_COMPOSITING_MODE` env vars. This **must** happen before `tauri::Builder::default()` — the env vars are read once at WebView creation.
- **Window flash prevention**: Window starts `visible: false` in `tauri.conf.json`. Frontend calls `show()` via `revealWindow()` only after first paint (theme applied). Don't change this — the setting restore plugin (`window-state`) is configured to skip visibility flags.
- **Vendor files**: `src/vendor/mathjax/` and `src/vendor/mermaid/` are gitignored (generated by `npm postinstall`). After `npm install`, the app is fully offline except LAN-shared pages (which load MathJax/Mermaid from CDN).
- **Math delimiters**: Comrak parses `$...$`/`$$...$$`. Pandoc-style `\(...\)`/`\[...\]` are preprocessed to `$...$` before Comrak. ` ```math ` fenced blocks become `$$...$$`.
- **No linter, formatter, or typecheck config** exists. No CI runs lint/format checks.

## Settings

Persisted to `~/.config/emede/settings.json`. All fields optional with defaults in `settings.rs`. GPU acceleration defaults to `false`, Mermaid diagrams default to `true`.

## Release workflow

`.github/workflows/release.yml` triggers on `v*` tags. Uses `tauri-apps/tauri-action@v0` to build deb/rpm; also compresses binary + icon into `emede-v<VERSION>-x86_64-linux.tar.gz` with SHA256 checksum attached to the release.
